# Teacher Portal - APKL

The teacher portal allows educators to manage their courses, assignments, and grades.

## Features

- **Teacher Registration & Login** - Secure authentication
- **My Courses** - Manage all assigned courses
- **Assignments** - Create, manage, and track assignments
- **Grades** - Input and manage student grades
- **Students** - View enrolled students
- **Announcements** - Post announcements to students
- **Dashboard** - Overview of teaching activities

## Folder Structure

```
teacher-dashboard/
├── index.html (teacher dashboard)
├── style.css (styling)
└── script.js (functionality)
```

## Files

- `teacher.html` - Main teacher portal (registration + dashboard combined)
- Located in the main APKL folder

## Access

Teachers can access the portal at: `teacher.html`
